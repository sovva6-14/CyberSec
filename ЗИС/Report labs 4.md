<h1 style="text-align:center">Отчет по лабораторной работе №4</h1>

<h2>Задание: </h2>

- Скачать из облачного хранилища многотомный архив образа виртуальной машины с
предустановленной DLP-системой.
- Разархивировать многотомный архив и получить целостный файл образа виртуальной
машины.
- Подключить полученный образ к выбранной системе виртуализации и создать
виртуальную машину.
- Запустить виртуальную машину с DLP-системой и выполнить в ней три лабораторные
работы, направленные на изучение принципов функционирования и базовых операций
администрирования DLP-системы (состав и содержание лабораторных работ задаются
преподавателем).
- Подробно задокументировать ход выполнения каждой лабораторной работы,
сопровождая описание пояснениями и скриншотами ключевых действий в интерфейсе
DLP-системы и операционной системы.
- По каждой из трёх лабораторных работ сформулировать выводы с акцентом на
освоенные функции DLP-системы и практическое значение выполненных действий.
- Подготовить итоговый отчёт по результатам всей работы.


<h2>Выполнение: </h2>

Скачать необходимые архивы. После того, как все файлы были разархивированны, необходимо установить Windows 2003 Server на VirtualBox. После установки, запустить машину и войти под пользователем "Администратор", пароль "13". После успешного входа необходимо сменить пароль на учетной записи Администратор. Нужно нажать кнопку Пуск -> Настройка -> Панель управления -> Администрирование -> Управление компьютером -> Пользователи -> Выбрать пользователя Администратор, нажать "Действие" и нажать "Задать пароль" и указать новый пароль для этого пользователя.

После необходимо перезагрузить компьюер. Нажать на кнопку Пуск -> В поле "Выберите желаемое действие" вместо Выключение, выбрать "Перезагрузка", ввести примечание (любое) и нажать кнопку "ОК".

Далее необходимо открыть папку "Консоли" на рабочем столе и выбрать программу "Search Server Console". Нажать на нее два раза левой кнопкой мыши, для того, чтобы она открылась.
